1) O que é Git?

R - Git é uma ferramenta para versionamento de arquivos. O git serve para que você possa criar uma linha do tempo com diferentes pontos das 
alterações feitas. Assim, você consegue manter o histórico das alterações de um mesmo arquivo ou arquivos ao longo do tempo.

2) Qual comando utilizamos para inicializar um repositório Git?

c) git init

3) Na lista abaixo qual opção não tem um comando git?

b) add, status, log, dif,

4) Qual comando utilizamos para comitar as alterações feitas no repositório e ainda escrever uma mensagem para o comite?

R - git commit -m "seu commit"

5) Qual o comando utilizamos para obter um novo projeto de um repositório?

c) clone

8) Quais comandos você utiliza para criar um novo branch e alterar do atual para o novo?

a) branch, checkout

9) O que é o Github?

R - Github é só um site no qual você pode fazer upload de projetos git. Github dá visibilidade da linha do tempo de diferentes versões do projeto, provê ferramentas de métricas (como quais usuários alteraram mais o projeto, quando alteraram, o que alteraram, etc), gerência de problemas relatados (tickets ou issues), dentre outras.

10) Quais comandos você utiliza para baixar e enviar atualizações para o repositório?

d) pull, push

11) Qual comando utilizamos para reescrever a mensagem do último commit?

R - git commit --amend
